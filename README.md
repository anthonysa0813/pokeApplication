# Pokemon App

![imagenpokemonApp](src/assets/images/pokeapp.png)

Antes de ejecutar el proyecto, se debe de instalar las dependencias.

```
npm install
```

Parra correr el proyecto

```
  npm run dev
  ó
  yarn dev
```

## Stack principal

- react redux toolkit
- react toastify
- sass
- react router dom v6
- react icons
- vite
